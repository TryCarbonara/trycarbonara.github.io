# Supported APIs

* [Cloud Identify API](https://trycarbonara.github.io/swagger.html): Calculates the energy consumption & carbon emission leveraging application usage, running on Kubernetes on cloud (aws, azure, or gcp)
* [Node Identify API](https://trycarbonara.github.io/node-swagger.html): Calculates the energy consumption & carbon emission leveraging direct power consumption (using sensors) and process usage, running on Linux (bare-metal) (OS >= Ubuntu 22.04 LTS)
